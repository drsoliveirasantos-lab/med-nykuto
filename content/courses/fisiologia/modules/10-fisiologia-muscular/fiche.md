# Ficha rápida — Fisiología muscular

## Idea central
El músculo transforma ATP en fuerza mecánica. El músculo esquelético y el cardíaco usan troponina C; el músculo liso usa calmodulina y MLCK.

## Tabla clave
| Tipo | Control | Estriaciones | Regulador |
|---|---|---|---|
| Esquelético | Voluntario | Sí | Troponina C |
| Cardíaco | Involuntario | Sí | Troponina C |
| Liso | Involuntario | No | Calmodulina / MLCK |

## Mecanismos
- Esquelético: potencial de acción → túbulo T → DHPR → RyR1 → Ca²⁺ → troponina C.
- Cardíaco: Ca²⁺ extracelular + retículo sarcoplásmico → meseta → periodo refractario largo.
- Liso: Ca²⁺ → calmodulina → MLCK → fosforilación de miosina.

## Clínica
Miastenia = receptor nicotínico. Botulismo = liberación de ACh. Organofosforados = AChE. Hipertermia maligna = RyR1/Ca²⁺. Rigidez cadavérica = falta de ATP.
